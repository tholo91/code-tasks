# code-tasks Wiki

Welcome to the **code-tasks** wiki! This project aims to bridge the gap between late-night inspiration and GitHub execution.

## The Vision
`code-tasks` is a "Quick Capture" PWA inspired by the sleek UX of "Things" and "Sorted 3." It provides a stable, trustworthy, and beautiful interface for managing ideas directly in a GitHub repository via a transparent Markdown file.

### Key Pillars
- **Frictionless Capture**: No extra accounts, just GitHub. Built for speed and one-handed use.
- **AI-Native**: Every task list is designed to be instantly actionable for AI agents.
- **Offline-First**: Reliable sync that ensures no spark of genius is lost to a bad connection.
- **High-Fidelity UX**: Beautiful animations and a minimal, focused interface.
